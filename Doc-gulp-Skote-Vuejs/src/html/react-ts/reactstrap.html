Overview Official Website

Reactstrap is a popular open-source library that provides a set of Bootstrap components implemented as pure React components. It allows you to easily integrate Reactstrap's styling and UI components into your React applications without the need for jQuery or additional JavaScript dependencies.

It is a required plugin. We suggest that do not remove this.
Usage
We've used reactstrap.
Add Package
yarn add reactstrap
Remove Package
yarn remove reactstrap ( you can remove package by removing specific package from package.json )
