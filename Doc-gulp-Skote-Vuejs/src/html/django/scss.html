Structure

Skote has managed the scss file in the ( src/scss ) folder. We have used gulp-sass for scss compilation. You can add your custom scss in the custom.scss file instead of doing changes in the app.scss or other scss files as it will create conflicts in future updates.

  • scss
    • Custom
      • components
        • All the components scss files are here.
      • fonts
        • _fonts.scss
      • pages
        • All the pages scss files are here.
      • plugins
        • icons
          • All the icons scss files are here
        • All the plugins scss files are here.
      • structure
        • _footer.scss // Footer related scss will be in this file only.
        • _layouts.scss // All the layouts related scss will be in this file only
        • _menu.scss
        • _page-head.scss
        • _topbar.scss
      _variables.scss
      _variables-dark.scss // includes all dark mode's variables
      app.scss // it has imported all other scss files and create app.min.css file.
      bootstrap.scss // it has imported all the bootstrap's scss files.
      icons.scss // includes all icons files.

Other SCSS file Structure

We have created all the scss files in the src/scss folder.

components
All components related scss files are created in this folder.
fonts
All fonts are imported in this scss file.
pages
All the pages related to scss are written in this folder and page wise scss created i.e. _authentication.scss, _chat.scss, _dashboard.scss etc
plugins
All the plugins ( Apex chart, ckeditor, full calendar etc ) are created in the plugins folder.
Quick customization of layout using scss.

You can make changes in the src/scss/_*.scss file.

Variables Description 
$primary:       $purple;
$secondary:     $blue;
$success:       $green;
$info:          $teal;
$warning:       $yellow;
$danger:        $red;
$light:         $gray-200;
$dark:          $gray-900;
 You can manage those colors to update the color theme. 
--#{$prefix}font-sans-serif:                         "Inter",sans-serif
--#{$prefix}font-monospace:                          "Inter",sans-serif
 You can set the default font in these variables.
