Sparkline ChartsOfficial Website

This jQuery plugin generates sparklines (small inline charts) directly in the browser using data supplied either inline in the HTML, or via javascript.

Javascript
<!-- ApexChart js -->
<script src="{{url_for('static' ,filename='libs/jquery-sparkline/jquery.sparkline.min.js')}}"></script>
Initjs (Custom js)
<!-- Init js -->
<script src="{{url_for('static' ,filename='js/pages/sparklines.init.js')}}"></script>
Add Package
yarn add jquery-sparkline --save
Remove Package
yarn remove jquery-sparkline --save or you can remove package by removing specific package from package.json
