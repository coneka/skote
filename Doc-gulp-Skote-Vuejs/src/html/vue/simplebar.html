Overview Official Website

SimpleBar does only one thing: replace the browser's default scrollbar with a custom CSS-styled one without losing performances.

Add Package
yarn add simplebar-vue --save
Import Package
import simplebar from 'simplebar-vue';
Remove Package
yarn remove simplebar-vue ( you can remove package by removing specific package from package.json )
