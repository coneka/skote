OverviewOfficial Website

Amchart library allows you to use the potential of Google Maps in a simple way. No more extensive documentation or large amount of code.

Add Package
yarn add @amcharts/amcharts4
Import Package
import * as am4core from '@amcharts/amcharts4/core';
import * as am4maps from '@amcharts/amcharts4/maps';
Remove Package
yarn remove @amcharts/amcharts4map ( Or you can remove package by removing specific package from package.json )
