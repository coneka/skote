RTL Mode

RTL Setup

To make the default RTL version just follow the below steps.

Update the below css file links in the templates/element/head-css.php file. 

<link href="/css/bootstrap-rtl.min.css" rel="stylesheet" type="text/css">
<link href="/css/app-rtl.min.css" rel="stylesheet" type="text/css">

and add the below attribute in the templates/element/main.php file to html tag.
dir="rtl" so you final html tag will be <html dir="rtl">
