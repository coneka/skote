Dark Mode Setting

You can simply change layout mode to Dark by doing the below changes in the html file /partials/main.html and laravel file resources/views/layouts/master.blade.php.

Simply set the attribute data-bs-theme="dark" to the body tag. It will update the layout mode to Dark mode. No need to change any css or js files.

Attribute Value
data-bs-theme 
<body lang="en" data-bs-theme="dark"">
