How to use pre-built layouts?

Each of the layout options is provided below with steps you would need to perform in file Admin/partials/body.php in body tag.

Light Sidebar & Topbar Dark
data-sidebar="dark"
data-topbar="light"
Compact Sidebar
data-sidebar="dark" data-sidebar-size="small"
Icon Sidebar
data-sidebar="dark" data-keep-enlarged="true"
class="vertical-collpsed"
Boxed Layout
data-sidebar="dark" data-keep-enlarged="true"
class="vertical-collpsed"
data-layout-size="boxed"
Colored Sidebar
data-sidebar="colored"
Scrollable layout
data-sidebar="dark" data-layout-scrollable="true"
