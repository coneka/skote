Dark Mode Setting

You can simply change layout mode to Dark by doing the below changes in the file Views/Shared/_Layout.cshtml.

Simply set the attribute data-layout-mode="dark" to the html tag. It will update the layout mode to Dark mode. No need to change any css or js files.

Attribute Value
data-layout-mode 
<!doctype html>
<html lang="en" data-layout-mode="dark">
