OverviewOfficial Website

Flatpickr is a robust and lightweight datetime picker solution.

Timepickr Examples:
Add Package
yarn add react-flatpickr
Import Package
import Flatpickr from "react-flatpickr";
Remove Package
yarn remove react-flatpickr ( or you can remove package by removing specific package from package.json )
Title React
Basic Timepicker 
<Flatpickr 
className="form-control"
placeholder="Select time"
options={{
enableTime: true,
noCalendar: true,
dateFormat: "H:i",
}}
/>
24-hour Time Picker 
<Flatpickr className="form-control"
options={{
enableTime: true,
noCalendar: true,
dateFormat: "H:i",
time_24hr: true
}}
/>
Time Picker w/ Limits 
<Flatpickr
className="form-control"
options={{
enableTime: true,
noCalendar: true,
dateFormat: "H:i",
minTime: "16:00",
maxTime: "22:30",
}}
/>
Inline 
<Flatpickr
className="form-control"
options={{
enableTime: true,
noCalendar: true,
inline: true,
}}
/>
