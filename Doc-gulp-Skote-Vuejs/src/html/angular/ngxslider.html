OverviewOfficial Website

NGX Slider is a lightweight range slider with multi-touch support and a ton of features.

Add Package
npm install --save ngx-slider-v2
Remove Package
npm uninstall --save ngx-slider-v2 or you can remove package by removing specific package from package.json
Examples:
Title Templates TypeScript
Basic Example 
<ngx-slider [(value)]="value" [options]="options"></ngx-slider>

value: number = 100;
options: Options = {
    floor: 0,
    ceil: 200
};
