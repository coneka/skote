Dark Mode Setting

You can simply change layout mode to Dark by doing the below changes in the file src/app/layout/layout-models.ts.

update the object parameter LAYOUT_MODE: dark. It will update the layout mode to Dark mode. No need to change any css or js files.

Attribute Value
data-bs-theme 
<!doctype html>
<html lang="en" data-bs-theme="dark">

Light Mode Setting

You can simply change layout mode to Light by doing the below changes in the file src/app/layout/layout-model.ts.

update the object parameter LAYOUT_MODE: light It will update the layout mode to Light mode. No need to change any css or js files.

Attribute Value
data-bs-theme 
<!doctype html>
<html lang="en" data-bs-theme="light">
