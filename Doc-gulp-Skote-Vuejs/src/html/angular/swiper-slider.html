Overview Official Website

Swiper is the most modern slider with hardware accelerated transitions and amazing native behavior.

Add Package
npm i ngx-owl-carousel-o
Add the swiper styles to the app styles in angular.json.
"styles" : [ "node_modules/ngx-owl-carousel-o/lib/styles/prebuilt-themes/owl.carousel.min.css"
"node_modules/ngx-owl-carousel-o/lib/styles/prebuilt-themes/owl.theme.default.min.css" , ]
Remove Package
npm uninstall ngx-owl-carousel-o or you can remove package by removing specific package from package.json
