OverviewOfficial Website

"ng-chartist" simplifies the integration of Chartist.js charts into Angular applications, making it easier to create interactive and responsive charts for your web projects.

Add Package
npm i chartist
npm i ng-chartist
Remove Package
npm uninstall chartist
npm i ng-chartist
you can remove package by removing specific package from package.json

Charts

You can refer to the src/app/pages/chart/chartist directory for various professional chart examples in your project.
